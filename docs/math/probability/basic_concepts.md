###  概率的公理化的定义

??? example "例：配对问题" 

	$n$ 人 $n$ 帽，随机取，问：无人拿对的概率。

	记：$A_i$ 为第 $i$ 个人拿到第 $i$ 顶帽子。 $P(A_i) = 1/n = \frac{(n-1)!}{n!}$
	
	考虑对立事件：至少有一个人拿到帽子。

	$$P(A_1 + \cdots +　A_n) = \sum_{i=1}^n P(A_i) + \cdots + (-1)^{r+1}\sum_{i_1< \cdots <i_r} P(A_{i_1} \cdots A_{i_r})$$
