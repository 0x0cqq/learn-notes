## Vue CLI

名称：`@vue/cli`

安装：

```bash
npm install -g @vue/cli
# OR
yarn global add @vue/cli
```

```bash
vue --version
```

**创建新项目：**`vue create hello-world`

**图形化界面：** `vue ui`

**安装插件** `@foo/vue-cli-plugin-bar`：`vue add @foo/bar`

**CLI 服务：**`npm run serve/yarn serve`

> 实际上是 `npx vue-cli-service serve/build/inspect`
