### 编译与解释

| 编译程序                                           | 解释程序                                     |
| -------------------------------------------------- | -------------------------------------------- |
| 把 高级程序设计语言 转换成 低级程序设计语言 的程序 | 直接接受并运行高级程序设计语言的程序。       |
| 编译和运行是两个独立分开的阶段。                   | 一条条获取源程序语句。一旦分析完就开始运行。 |
| 会生成目标代码，不支持交互环境                     | 不生成目标代码，支持交互环境                 |
